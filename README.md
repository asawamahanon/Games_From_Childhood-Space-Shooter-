# Games_From_Childhood-Space-Shooter-
ผลงานตัวนี้เป็นผลงานที่เกิดจากคิดถึงเกมในวัยเด็กน้อย ที่เล่นเกมจากมือถือพับของพี่สาวและเกมนั้นชื่อ SpaceInvader เวลาที่พี่ไม่ใช้มือถือเราก็จะขอเล่นมือถือของแกตลอด
ทำให้ตอนที่ได้ทำโจทย์เกมอีกครั้ง จึงได้เกิดเป็นผลงานตัวนี้ขึ้นมา โดย assets เกือบทั้งหมดนำมากจาก Unity  ไม่ว่าจะเป็น BackGround,Player,Enemy,LaserBeam ฯลฯ ยกเว้นตัวชื่อเกมและปุ่มต่างๆ
ผมนำทรัพยากรจาก assets ไปเข้า Canva เพื่อทำรูปชื่อเกมกับปุ่มเกม แล้วเพลงประกอบเกมก็เอาจากเว็บหาSoundเพลงอย่าง Pixabay โดยนำมาตั้งแต่ Effect 3 Effect(มี1Effectที่ไม่ได้ใช้งาน),BackGroundSound 2 Sound 
ในด้านของการcode หาดูจากyoutubeทั้งไทยและชาวต่างชาวผสมกับcodeในเกมที่แล้วอย่าง Ninjarun โดยนำฟังก์ชั่น Coroutine มาปรับใช้กับตัวงานนี้ด้วยเพื่อทำความเข้าใจกับตัวฟังก์ชันและความคุ้นเคยของฟังก์ชัน
